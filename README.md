### 重要
- サービスとしてコードを実行
- BLEのビーコンを飛ばす（プラグインは使えそうなやつをとりあえず落とした）
- ビーコンを拾った結果を端末に保存しておく→Hive
- ビーコン拾った結果を画面で集計して見る
